Numerot
=======

Convert numbers to finnish words. Only numbers -9 .. 9 are supported at this state :)

npm install js-numerot --save

## Usage

  var numerot = require('numerot');

  console.log(numerot(1));//yksi
  console.log(numerot(2));//kaksi
